Usa un System User Token de Business Manager. Los temporales de 24h solo sirven para pruebas.

Cómo obtener las credenciales

  1. developers.facebook.com → App → WhatsApp → API Setup
  2. Copiar Phone Number ID y WABA ID
  3. Generar Access Token desde un System User (Business Manager) con permisos whatsapp_business_messaging y whatsapp_business_management
  4. Configurar el webhook en Meta apuntando a https://tu-dominio/mensajeria/webhook/whatsapp/ usando el Verify Token de arriba
